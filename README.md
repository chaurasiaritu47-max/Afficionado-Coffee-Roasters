# Title: Product Optimization & Revenue Contribution Analysis for Afficionado Coffee Roasters

This project analyzes the sales data of Afficionado Coffee Roasters using Python, EDA, and data visualization techniques. A new Revenue column was created to measure total sales performance, and the analysis was used to identify top-selling products, revenue-generating categories, and store performance through charts and business insights.

An interactive Streamlit dashboard was developed to display KPI metrics, charts, filters, and product performance analysis in a user-friendly format. The dashboard includes custom styling, interactive filters, and visualizations that help users easily explore revenue trends, product demand, and overall business performance.
Live app link: https://afficionado-coffee-roasters-obfmftkxlcgz9x4p5r7axt.streamlit.app/
